# NekoServe v0.1.0 — Initial Release

Released: 2026-04-13

First public release of NekoServe, a desktop cat-café discrete-event simulation for classroom demonstration.

## Features

- **Simulation core** — discrete-event simulation powered by Python SimPy 4; full customer lifecycle (arrive → queue → seated → order → dine → cat interaction → leave); Poisson arrivals and normal-distributed service times; cat rest mechanic; reproducible via random seed.
- **UI pages** — simulation settings (13 parameters + 3 preset scenarios), results (10 KPIs + 3 charts), event log with type filter and search, about page with classroom context.
- **Scenario comparison** — run multiple configurations and compare KPIs side-by-side (up to 3 runs).
- **Custom scenario presets** — save, name, and persist your own parameter sets across restarts.
- **Exports** — JSON for full results, CSV for metrics, CSV for filtered event log.
- **Learning sidebar** — collapsible classroom-notebook-style content, context-aware per page.
- **Native desktop polish** — macOS `hiddenInset` title bar with traffic lights integrated into the header, custom app icon, custom About dialog, window-state persistence.

## Tech stack

Electron 33, React 18, TypeScript 5, Tailwind CSS, Recharts, Python 3.11 + SimPy 4 (bundled via PyInstaller), electron-vite, electron-builder.
