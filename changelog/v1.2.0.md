# NekoServe v1.2.0 -- Interactive learning, formula explainer, and beginner/expert differentiation

Released: 2026-04-18

v1.2.0 transforms the How-It-Works page into an interactive learning experience and adds clickable formula explanations across the Results page. A floating Beginner/Expert toggle makes the entire app accessible to math-phobic students while preserving full technical depth for advanced users.

## Interactive How-It-Works page

- **7 self-contained mini-demos** replacing the previous static essay format:
  - **FEL Stepper**: step through a discrete-event timeline with emoji flow and staggered labels
  - **Resource Queue**: add/release customers with a capacity slider, emoji queue visualization
  - **Patience Race**: animated dual progress bars racing seat availability vs customer patience
  - **Cat Pool Dynamics**: coin-flip animation with countdown rest timers
  - **Accumulator Watch**: step through 5 customers building up metrics, then aggregate
  - **Seed Scatter**: SVG bar chart with adjustable replication count and re-roll button
  - **CI Explorer**: number-line visualization with N/confidence-level controls + recharts sensitivity chart

- **Collapsible code blocks** with Python syntax highlighting (keywords, builtins, strings, comments, numbers, operators) and a hover-to-reveal copy button

- **Section card layout** with IntersectionObserver slide-in animations, each card containing concept text, interactive demo, takeaway note, and collapsible code

## Beginner/Expert differentiation

- **Floating level toggle** (fixed bottom-right) on both How-It-Works and Results pages, always accessible while scrolling
- **Fully separate concept text** for each mode: beginner uses life analogies ("fast-forwarded movie," "rolling dice," "highway traffic jam"), expert uses full DES terminology (FEL, Wq, Erlang-C, t-distribution)
- **Beginner mode hides code blocks** entirely, showing only interactive demos and plain-language takeaways
- **Demo-level enhancements** for beginner mode: emoji step flow, status bubbles ("Full! 🙈"), larger emoji feedback, guide text prompts, progress bar visuals
- **Separate takeaway notes** per level: beginner uses colloquial phrasing without English terms

## Interactive formula explainer

- **New `InteractiveFormula` component**: renders the full KaTeX formula with a row of clickable symbol chips below it
- **26 symbol explanations** across 4 Results page formula sections (Flow, Wait, Utilization, Cat), each with:
  - Plain-language name and description using real-world analogies
  - Unit label
  - Concrete numeric example in an amber highlight box
- All explanations available in both zh-TW and en

## Bug fixes

- Fixed double-render of confidence interval formula in learning content (BlockMath wrapped in redundant Formula component)
- Removed stale WhatsNewModal ("update complete" popup) from App.tsx
